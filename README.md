# Dmujeres-Traccar

Proyecto de hard-fork de Traccar Web, optimizado con backend en Go + Fiber y frontend React optimizado con Google Maps Carreteras.

## Arquitectura y Guía Scrum
Consulta la guía completa de arquitectura y desarrollo en la documentación oficial.

## Ejecutar en desarrollo
1. **Backend (Go):**
   ```bash
   cd backend
   go run cmd/server/main.go
   ```
2. **Frontend (React):**
   ```bash
   cd frontend
   npm run dev
   ```
